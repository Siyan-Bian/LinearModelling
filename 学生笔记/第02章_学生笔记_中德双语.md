## 第 2 章：简单线性回归 / Kapitel 2: Das einfache lineare Regressionsmodell

### 2.1 学习目标 / Lernziele

**中文**：本章建立最简单的线性模型——一个 $Y$，一个 $x$，一条直线。我们要解决五个问题，按由浅入深的顺序排列：(1) 模型长什么样，假设是什么？(2) 怎么画"最优"直线（最小二乘）？(3) 估计出来的参数有什么性质，可靠吗？(4) $R^2$ 和平方和分解怎么衡量拟合好坏？(5) 怎么预测新的观测？什么时候线性近似会出问题？

**Deutsch**: Dieses Kapitel behandelt das einfachste lineare Modell — ein $Y$, ein $x$, eine Gerade. Fünf Fragen werden beantwortet: (1) Wie sieht das Modell aus und welche Annahmen werden getroffen? (2) Wie findet man die „beste" Gerade (Kleinste Quadrate)? (3) Welche Eigenschaften haben die Schätzer, wie zuverlässig sind sie? (4) Wie misst man die Anpassungsgüte mit $R^2$ und Quadratsummenzerlegung? (5) Wie prognostiziert man neue Beobachtungen und wann versagt die lineare Näherung?

### 2.2 知识点 1：模型与假设——"一条直线 + 随机抖动" / Modell und Annahmen

#### 生活比喻先行

中文：想象你考驾照练习刹车。教练说"速度越快，刹车距离越长"。你做了 50 次实验，记下每次的速度 $x_i$ 和刹车距离 $Y_i$。你发现：同样开 20 mph，每次刹车距离不完全一样（路面、反应时间、风都不同），但**平均来看**确实更快→距离更长。简单线性回归做的就是：**用一条直线来描述"平均趋势"，同时承认每次实验围绕平均有随机波动**。

Deutsch: Stell dir vor, du übst Bremsen für die Fahrprüfung. Bei gleicher Geschwindigkeit ist die Bremsdistanz jedes Mal etwas anders (Straße, Reaktionszeit, Wind). Aber **im Mittel** gilt: mehr Geschwindigkeit → mehr Bremsdistanz. Einfache lineare Regression beschreibt diesen **mittleren Trend** mit einer Geraden und akzeptiert die zufällige Streuung.

#### 模型公式（Folie 18）

$Y_i = \beta_0 + \beta_1 x_i + \epsilon_i, \quad i = 1, \dots, n$

| 符号 | 含义 中文 | Bedeutung Deutsch |
|---|---|---|
| $Y_i$ | 第 $i$ 个观测的目标变量（随机的）| Zielgröße der $i$-ten Beobachtung (zufällig) |
| $x_i$ | 第 $i$ 个观测的解释变量（**固定已知**）| Einflussgröße (fest und bekannt) |
| $\beta_0$ | 截距（未知参数）| Achsenabschnitt (unbekannter Parameter) |
| $\beta_1$ | 斜率（未知参数）| Steigung (unbekannter Parameter) |
| $\epsilon_i$ | 随机误差（唯一的随机来源）| Zufallsfehler (einzige Zufallsquelle) |
| $n$ | 样本量 | Anzahl der Beobachtungen |

#### 假设的"洋葱层"

中文：讲义把假设分层展示，像洋葱一样，从最弱到最强。每一层都给出额外的结论，所以你必须知道"我用的结论依赖哪一层假设"。

| 层次 | 假设 | 给出什么结论 |
|---|---|---|
| 第 1 层（最弱）| $E(\epsilon_i) = 0$ | $\hat\beta_0$、$\hat\beta_1$ 无偏 |
| 第 2 层 | + $V(\epsilon_i) = \sigma^2$（同方差）+ 独立 | 方差公式 $V(\hat\beta_1)$、$V(\hat\beta_0)$ |
| 第 3 层（最强）| + $\epsilon_i \sim N(0, \sigma^2)$ | $t$ 分布、精确置信区间、KQ = ML |

Deutsch: Die Annahmen sind geschichtet:

| Schicht | Annahme | Daraus folgt |
|---|---|---|
| 1 (schwächste) | $E(\epsilon_i) = 0$ | Unverzerrtheit von $\hat\beta_0$, $\hat\beta_1$ |
| 2 | + $V(\epsilon_i) = \sigma^2$ (Homoskedastizität) + Unabhängigkeit | Varianzformeln |
| 3 (stärkste) | + $\epsilon_i \sim N(0, \sigma^2)$ | $t$-Verteilung, exakte KI, KQ = ML |

#### 为什么 $x_i$ 是固定的？

中文：讲义把 $x_i$ 当作已知常数（不是随机变量）。这简化了所有推导——$\hat\beta_1$ 的随机性全来自 $\epsilon_i$。在实验设计中（比如你控制车速），$x_i$ 确实可控；在观测数据中，$x_i$ 可能也是随机的，但可以通过**条件于 $x_i$** 来理解，结论在条件下仍然成立。

Deutsch: Die $x_i$ werden als fest betrachtet. Das vereinfacht die Herleitung: Alle Zufälligkeit von $\hat\beta_1$ kommt aus $\epsilon_i$. In Experimenten kontrolliert man $x_i$ tatsächlich; bei Beobachtungsdaten kann man die Ergebnisse **bedingt auf $x_i$** interpretieren.

#### R 练习 / R-Übung

```r
data(cars)
# 50 Beobachtungen: speed (mph) und dist (Bremsdistanz in ft)
str(cars)
plot(dist ~ speed, data = cars,
     main = "Bremsdistanz vs. Geschwindigkeit",
     xlab = "Geschwindigkeit (mph)", ylab = "Bremsdistanz (ft)")

fit_cars <- lm(dist ~ speed, data = cars)
abline(fit_cars, col = "red", lwd = 2)
coef(fit_cars)
# Intercept ≈ -17.6, Slope ≈ 3.93
# 解释：速度每增加 1 mph，平均刹车距离增加约 3.93 ft
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：截距 $\hat\beta_0 \approx -17.6$ 意味着速度为 0 时刹车距离是负的。这有意义吗？
   Deutsch: $\hat\beta_0 \approx -17{,}6$ bedeutet eine negative Bremsdistanz bei Geschwindigkeit $0$. Ist das sinnvoll?
   **答案 / Antwort**：$x = 0$ 不在数据合理范围内（最小速度是 4 mph），所以截距只是数学上使直线最优的值，不应做实际解释。讲义 Folie 27 也强调"in vielen Fällen irrelevant"。

2. 中文：假设误差方差随速度增大而增大（高速时散布更大），第 2 层假设还满足吗？
   Deutsch: Wenn die Fehlervarianz mit der Geschwindigkeit zunimmt, ist die Homoskedastizitätsannahme noch erfüllt?
   **答案 / Antwort**：不满足。$V(\epsilon_i) = \sigma^2$ 要求所有 $i$ 方差相同。如果方差不齐，KQ 仍可用但方差公式和 $t$ 推断可能不可靠——第 8 章（加权最小二乘）会解决这个问题。

### 2.3 知识点 2：怎么画"最优"直线？——成本函数的选择 / Wie findet man die beste Gerade? — Wahl der Kostenfunktion

#### 问题：无穷多条直线，选哪条？

中文：散点图上可以画无穷多条直线。要选"最好的一条"，必须先定义"什么叫好"。讲义 Folie 19–25 用图形展示了两种思路。

Deutsch: Durch eine Punktwolke kann man unendlich viele Geraden legen. Um die „beste" auszuwählen, muss man erst definieren, was „gut" bedeutet. Folien 19–25 zeigen zwei Ansätze grafisch.

#### 思路 A：绝对距离（Folie 20）

中文：每个数据点到直线的垂直距离取绝对值，然后求总和：

$\text{Cost}_{\text{abs}} = \sum_{i=1}^n |Y_i - \beta_0 - \beta_1 x_i|$

图形上，每个残差画成一根**竖线**（长度 = 绝对距离），我们要让所有竖线总长度最短。

优点：对离群点较稳健（距离变成 2 倍，成本也只变 2 倍）。缺点：绝对值函数在 0 处不可微，数学处理不方便，没有唯一闭式解。

Deutsch: Jeder Datenpunkt hat einen vertikalen Abstand zur Geraden; man summiert die Beträge. Grafisch ist jeder Abstand eine **Strecke**. Vorteil: robuster gegen Ausreißer. Nachteil: nicht differenzierbar bei $0$, keine geschlossene Formel.

#### 思路 B：平方距离——最小二乘（Folie 21）

中文：每个残差取平方，然后求总和：

$\text{Cost}_{\text{sq}} = \sum_{i=1}^n (Y_i - \beta_0 - \beta_1 x_i)^2$

图形上，每个残差画成一个**正方形**（边长 = |残差|，面积 = 残差²），我们要让所有正方形总面积最小。讲义 Folie 21 的彩色方块就是这个意思——大的残差对应大正方形，面积增长更快。

优点：处处可微，有唯一闭式解（正规方程），与正态误差下最大似然一致。缺点：对离群点敏感（距离变成 2 倍，面积变成 4 倍）。

Deutsch: Jedes Residuum wird quadriert und die Quadrate werden summiert. Grafisch ist jedes Residuum ein **Quadrat** (Seitenlänge = |Residuum|, Fläche = Residuum²). Die bunten Quadrate auf Folie 21 veranschaulichen dies — große Residuen erzeugen überproportional große Flächen. Vorteil: überall differenzierbar, eindeutige geschlossene Lösung, entspricht ML unter Normalverteilung. Nachteil: empfindlich gegenüber Ausreißern.

#### 为什么课程选择平方距离？

中文：讲义 Folie 22 提到 Gauß（约 1795 年）和 Legendre（1805 年）最早使用此方法。选择平方距离的核心原因是：

(a) 数学上方便——可以对 $\beta_0$、$\beta_1$ 求偏导并令为零，得到两个线性方程（正规方程），直接解出闭式公式。

(b) 统计上自然——如果误差正态分布，最小二乘就是最大似然（Folie 31/33 会证明），这意味着在所有估计量中它有最优的统计性质。

(c) 后续全部建立在此基础上——$R^2$、$F$ 检验、$t$ 检验都是围绕平方和展开的。

Deutsch: Die Vorlesung wählt quadratische Distanz, weil (a) die Optimierung durch Ableiten gelöst werden kann (Normalgleichungen), (b) unter Normalverteilung KQ = ML, und (c) Quadratsummenzerlegung, $R^2$, $F$- und $t$-Tests darauf aufbauen.

#### 讲义的动画图解（Folie 24：三张图的过渡）

中文：Folie 24 展示了三个阶段的图：

阶段 1：一条水平线（只用均值 $\bar Y$），方块很大——拟合很差。

阶段 2：一条有斜率但不是最优的线，方块大小混合。

阶段 3：最小二乘线——所有方块的总面积最小。

你无法让所有方块面积为零（除非点全在一条直线上），但最小二乘保证了在所有直线中，这条线让总面积最小。

Deutsch: Folie 24 zeigt drei Stadien: eine horizontale Linie (nur $\bar Y$, große Quadrate), eine schräge, aber nicht optimale Linie, und die KQ-Gerade mit minimaler Gesamtfläche.

#### R 练习：感受离群点对平方损失的影响 / R-Übung

```r
# 原始数据拟合
fit1 <- lm(dist ~ speed, data = cars)
coef(fit1)

# 加一个离群点：speed = 25, dist = 300（非常远）
cars2 <- rbind(cars, data.frame(speed = 25, dist = 300))
fit2 <- lm(dist ~ speed, data = cars2)
coef(fit2)

# 比较：斜率和截距都变了
plot(dist ~ speed, data = cars2,
     main = "Ausreißereffekt auf KQ-Gerade")
abline(fit1, col = "blue", lwd = 2, lty = 2)
abline(fit2, col = "red", lwd = 2)
legend("topleft",
       legend = c("ohne Ausreißer", "mit Ausreißer"),
       col = c("blue", "red"), lwd = 2, lty = c(2, 1))
# 一个离群点就能把直线"拽"过去——平方损失放大了大残差的影响
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：如果一个残差从 5 变成 10，绝对损失增加多少？平方损失增加多少？
   Deutsch: Wenn ein Residuum von $5$ auf $10$ steigt, um wie viel steigt die absolute Distanz? Um wie viel die quadratische?
   **答案 / Antwort**：绝对损失增加 $10 - 5 = 5$；平方损失增加 $100 - 25 = 75$。平方损失对大残差的"惩罚"强得多。

2. 中文：如果你的数据里有明显的录入错误（比如某个刹车距离被记成了 1000 ft），最小二乘直线会受到很大影响吗？
   Deutsch: Wenn eine Bremsdistanz fälschlich als $1000$ ft eingetragen wurde, wird die KQ-Gerade stark beeinflusst?
   **答案 / Antwort**：是的。这就是平方损失的缺点——一个极端值的残差被平方后非常大，会强烈拉偏直线。应该先检查数据质量。

### 2.4 知识点 3：估计公式与参数解释 / Schätzformeln und Interpretation

#### KQ 估计量的公式（Folie 23, 26）

中文：定义 KQ 估计量：

$(\hat\beta_0, \hat\beta_1) = \arg\min_{\beta_0, \beta_1} \sum_{i=1}^n (Y_i - \beta_0 - \beta_1 x_i)^2$

只要 $x$ 有变异（即不是所有 $x_i$ 都相同），估计**唯一存在**：

$\hat\beta_1 = \frac{S_{xY}}{S_x^2} = \frac{\frac{1}{n}\sum_{i=1}^n (x_i - \bar x)(Y_i - \bar Y)}{\frac{1}{n}\sum_{i=1}^n (x_i - \bar x)^2}$

$\hat\beta_0 = \bar Y - \hat\beta_1 \bar x$

Deutsch: Der KQ-Schätzer existiert und ist eindeutig, falls $\sum_{i=1}^n (x_i - \bar x)^2 \neq 0$:

$\hat\beta_1 = \frac{S_{xY}}{S_x^2}, \qquad \hat\beta_0 = \bar Y - \hat\beta_1 \bar x$

#### 用日常语言理解公式

中文：$\hat\beta_1$ 的公式可以拆成两部分理解：

**分子** $S_{xY}$：$x$ 和 $Y$ 的样本协方差——衡量"$x$ 偏高时 $Y$ 是否也偏高"。如果大的 $x$ 总是配大的 $Y$，分子为正；反之为负。

**分母** $S_x^2$：$x$ 的样本方差——衡量"$x$ 本身有多少变异"。变异越大，我们对趋势判断越有信心。

所以 $\hat\beta_1 = \frac{\text{x 和 Y 如何共同变化}}{\text{x 自己有多少变化}}$。

$\hat\beta_0 = \bar Y - \hat\beta_1 \bar x$ 说明回归线必过点 $(\bar x, \bar Y)$——这是回归的"锚点"。

Deutsch: $\hat\beta_1 = \frac{\text{gemeinsame Variation von } x \text{ und } Y}{\text{Variation von } x \text{ allein}}$. Die Gerade geht immer durch $(\bar x, \bar Y)$.

#### 正规方程：两个"正交条件"（Folie 26）

中文：对目标函数求偏导并令为零，得到**正规方程**（Normalgleichungen）：

$\sum_{i=1}^n \hat\epsilon_i = 0 \qquad \text{(残差之和为零)}$

$\sum_{i=1}^n \hat\epsilon_i \cdot x_i = 0 \qquad \text{(残差与 x 的加权和为零)}$

第一个方程保证残差"正负抵消"——直线不会系统性地偏上或偏下。第二个方程保证残差与 $x$ "不相关"——直线已经充分利用了 $x$ 的信息。

Deutsch: Die Normalgleichungen garantieren: (1) die Residuen summieren sich zu Null, und (2) die Residuen sind mit $x$ „unkorreliert" — die Gerade hat die Information von $x$ voll ausgenutzt.

#### 参数解释（Folie 27–28）

中文：**$\beta_1$ 的解释**：当 $x$ 增加 1 个单位时，$Y$ **平均**增加 $\beta_1$ 个单位。注意关键词是"平均"——不是每个个体都恰好增加 $\beta_1$。

生活例子：在 `cars` 数据中，$\hat\beta_1 \approx 3.93$，意思是速度每增加 1 mph，刹车距离**平均**增加约 3.93 ft。但某次实验可能增加 2 ft，另一次增加 6 ft——这就是 $\epsilon_i$ 的随机波动。

**$\beta_1$ 的大小依赖单位**：如果速度从 mph 换成 km/h（1 mph ≈ 1.609 km/h），新斜率大约是 $3.93 / 1.609 \approx 2.44$ ft/km·h⁻¹。模型描述的实际关系没变，只是数字变了。

**$\beta_0$ 的解释**：$x = 0$ 时 $Y$ 的平均值。但很多时候 $x = 0$ 不在合理范围内（速度为 0 时"刹车距离"无意义），所以 $\beta_0$ 往往只是数学上让直线位置正确的常数，不需要实际解释。

**$\sigma$ 的解释**（Folie 28）：误差的标准差，衡量数据点围绕回归线的**典型散布距离**。$\sigma$ 小 = 点紧紧围绕直线；$\sigma$ 大 = 点散布很开。

Deutsch: $\beta_1$ beschreibt die mittlere Änderung von $Y$ bei einer Einheit mehr in $x$ — **im Durchschnitt**, nicht für jede einzelne Beobachtung. $\beta_0$ ist der Erwartungswert bei $x = 0$, oft inhaltlich irrelevant. $\sigma$ misst die typische Streuung der Punkte um die Gerade.

#### R 练习：手动 vs. lm() / R-Übung

```r
x <- cars$speed
y <- cars$dist

# 手动计算 / Manuelle Berechnung
b1 <- sum((x - mean(x)) * (y - mean(y))) / sum((x - mean(x))^2)
b0 <- mean(y) - b1 * mean(x)
c(b0_manual = b0, b1_manual = b1)

# lm() 验证
coef(lm(dist ~ speed, data = cars))

# 验证正规方程 / Normalgleichungen prüfen
res <- y - (b0 + b1 * x)
c(sum_res = sum(res), sum_res_x = sum(res * x))
# 两个都应接近 0

# 回归线过 (x̄, Ȳ) / Gerade durch (x̄, Ȳ)
b0 + b1 * mean(x)   # 应该等于 mean(y)
mean(y)
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：如果所有人开的速度都是 15 mph（$x_i$ 全相同），能估计斜率吗？
   Deutsch: Kann man die Steigung schätzen, wenn alle $x_i = 15$?
   **答案 / Antwort**：不能。$\sum (x_i - \bar x)^2 = 0$，分母为零。直觉上，所有人速度一样，根本无法判断"速度变化如何影响距离"。

2. 中文：$\hat\beta_1$ 可以是负数吗？什么时候会是负的？
   Deutsch: Kann $\hat\beta_1$ negativ sein? Wann?
   **答案 / Antwort**：可以。当 $x$ 大时 $Y$ 倾向于小（负相关），比如"学习时间 $x$ 越长，考试错误数 $Y$ 越少"。

### 2.5 知识点 4：估计量的统计性质——为什么 KQ 估计可靠？ / Eigenschaften des KQ-Schätzers

#### 为什么要关心"性质"？

中文：我们算出了一个数字 $\hat\beta_1 = 3.93$。但这是这组特定数据的结果——如果重新做 50 次刹车实验，每次都会得到不同的 $\hat\beta_1$。所以 $\hat\beta_1$ 本身是一个**随机变量**，有自己的期望、方差和分布。搞清楚这些，才能判断"3.93 离真实值可能有多远"。

Deutsch: $\hat\beta_1 = 3{,}93$ ist das Ergebnis **dieser** Stichprobe. Bei neuen Daten erhält man einen anderen Wert. Daher ist $\hat\beta_1$ eine **Zufallsvariable** mit Erwartungswert, Varianz und Verteilung. Nur so kann man beurteilen, wie nah der Schätzwert am wahren Wert liegt.

#### 性质 1：无偏性（Folie 29, 32）——只需要 $E(\epsilon_i) = 0$

中文：如果 $E(\epsilon_i) = 0$，那么

$E(\hat\beta_0) = \beta_0, \qquad E(\hat\beta_1) = \beta_1$

意思是：虽然每次实验 $\hat\beta_1$ 的数值不同，但**平均来看**（如果你能无穷次重复实验）$\hat\beta_1$ 的均值恰好等于真实斜率 $\beta_1$。它没有系统性的偏差。

Deutsch: Unter $E(\epsilon_i) = 0$ gilt $E(\hat\beta_1) = \beta_1$ — im Mittel über viele Wiederholungen trifft der Schätzer den wahren Wert. Es gibt keinen systematischen Fehler.

#### 证明的关键洞察（Folie 32）：$\hat\beta_1$ 是 $Y_i$ 的加权和

中文：证明的核心步骤是把 $\hat\beta_1$ 写成

$\hat\beta_1 = \sum_{i=1}^n w_i Y_i, \qquad w_i = \frac{x_i - \bar x}{\sum_{j=1}^n (x_j - \bar x)^2}$

注意权重 $w_i$ 只依赖 $x_i$（固定的），不依赖 $Y_i$（随机的）。所以取期望时，$x_i$ 相关的权重可以直接提出来，期望作用在 $Y_i = \beta_0 + \beta_1 x_i + \epsilon_i$ 上，$E(\epsilon_i) = 0$ 后恰好得到 $\beta_1$。

这个"$\hat\beta_1$ 是 $Y_i$ 的线性组合"的事实非常重要，后面还有三个用途：(a) 传递正态性（$Y_i$ 正态 → $\hat\beta_1$ 正态），(b) 计算方差（利用独立性），(c) 理解每个数据点对估计的贡献（$x_i$ 离 $\bar x$ 越远，权重越大）。

Deutsch: Der Schlüssel ist: $\hat\beta_1 = \sum w_i Y_i$ mit festen Gewichten $w_i = (x_i - \bar x) / \sum (x_j - \bar x)^2$. Da die Gewichte fest sind, wirkt $E$ nur auf $Y_i$. Dieses „$\hat\beta_1$ ist eine Linearkombination der $Y_i$"-Resultat wird auch für Normalverteilungsübertragung, Varianzberechnung und das Verständnis des Einflusses einzelner Punkte gebraucht.

#### 性质 2：方差（Folie 30）——需要独立同方差

中文：若 $V(\epsilon_i) = \sigma^2$（同方差）且 $\epsilon_i$ 独立，则

$V(\hat\beta_1) = \frac{\sigma^2}{\sum_{i=1}^n (x_i - \bar x)^2}$

$V(\hat\beta_0) = \sigma^2 \left(\frac{1}{n} + \frac{\bar x^2}{\sum_{i=1}^n (x_i - \bar x)^2}\right)$

**直觉理解 $V(\hat\beta_1)$ 的三个因素**：

(a) **$\sigma^2$ 越大，$V(\hat\beta_1)$ 越大**——如果数据点围绕直线散布很开，你对斜率的判断自然不确定。想象在浓雾中判断一条路的坡度——看不清就不确定。

(b) **$n$ 越大，分母越大，$V(\hat\beta_1)$ 越小**——数据越多，估计越精确。

(c) **$x$ 的变异范围越大，$\sum (x_i - \bar x)^2$ 越大，$V(\hat\beta_1)$ 越小**——这很直觉：如果你只在 14–16 mph 之间测刹车，很难判断坡度；但如果你从 5 到 25 mph 都测了，趋势就清楚多了。

Deutsch: $V(\hat\beta_1)$ wird kleiner, wenn (a) die Fehlerstreuung $\sigma^2$ kleiner ist, (b) mehr Beobachtungen $n$ vorhanden sind, und (c) die $x$-Werte stärker variieren.

#### 性质 3：在正态误差下，KQ = ML（Folie 31, 33）

中文：如果 $\epsilon_i \sim N(0, \sigma^2)$，可以写出似然函数（Likelihood）。因为 $\beta_0$ 和 $\beta_1$ 只通过 $\sum \epsilon_i^2$ 出现在对数似然中，最大化似然就等价于最小化 $\sum \epsilon_i^2$。所以**在正态误差下，KQ 估计量 = ML 估计量**。

但 ML 对 $\sigma^2$ 的估计是 $\hat\sigma^2_{ML} = \frac{1}{n}\sum \hat\epsilon_i^2$（除以 $n$），它是**有偏的**——系统性地偏小。因此实践中使用**无偏版本**（除以 $n - 2$，见下节）。

Deutsch: Unter $\epsilon_i \sim N(0, \sigma^2)$ stimmt KQ mit ML überein. Aber der ML-Schätzer für $\sigma^2$ teilt durch $n$ und ist verzerrt. Daher verwendet man die unverzerrte Version mit $n - 2$.

### 2.6 知识点 5：$\sigma^2$ 的估计、$t$ 分布与置信区间 / Schätzung von $\sigma^2$, $t$-Verteilung und Konfidenzintervalle

#### $\sigma^2$ 的无偏估计（Folie 34）

$\hat\sigma^2 = \frac{1}{n-2}\sum_{i=1}^n \hat\epsilon_i^2$

中文：为什么除以 $n - 2$ 而不是 $n$？

生活比喻：你有 $n$ 个数据点，但你已经用掉了 2 个"自由度"去估计 $\beta_0$ 和 $\beta_1$。就像你有一笔预算，花了一部分去买截距和斜率的"信息"，剩下的才能用来估计误差大小。更严格地说，残差满足两个约束（正规方程 $\sum \hat\epsilon_i = 0$ 和 $\sum x_i \hat\epsilon_i = 0$），所以只有 $n - 2$ 个残差是"自由的"。

Deutsch: Man teilt durch $n - 2$, weil $2$ Freiheitsgrade für $\beta_0$ und $\beta_1$ „verbraucht" wurden. Die Residuen erfüllen $2$ Gleichungen (Normalgleichungen), also sind nur $n - 2$ Residuen frei bestimmbar.

**$\hat\sigma$（即 $\sqrt{\hat\sigma^2}$）的直觉**：它估计数据点围绕回归线的典型距离。R 里叫 `Residual standard error`。

#### $\chi^2$ 分布和独立性（Folie 35）

中文：在正态误差下，

$\frac{1}{\sigma^2}\sum_{i=1}^n \hat\epsilon_i^2 \sim \chi^2_{n-2}$

并且 $(\hat\beta_0, \hat\beta_1)$ 与 $\hat\sigma^2$ **相互独立**。

这个独立性非常关键——它是构造 $t$ 统计量的基础。$t$ 分布的定义就是"标准正态 ÷ 独立的 $\chi^2$ 的平方根"。

Deutsch: Unter Normalverteilung ist $\frac{1}{\sigma^2}\sum \hat\epsilon_i^2 \sim \chi^2_{n-2}$, und $(\hat\beta_0, \hat\beta_1)$ ist von $\hat\sigma^2$ unabhängig. Diese Unabhängigkeit ist entscheidend für die Konstruktion der $t$-Statistik.

#### $t$ 分布与置信区间（Folie 36–38）

中文：在正态误差下，

$\frac{\hat\beta_1 - \beta_1}{\hat\sigma_{\hat\beta_1}} \sim t_{n-2}$

其中 $\hat\sigma_{\hat\beta_1} = \sqrt{\frac{\hat\sigma^2}{\sum_{i=1}^n (x_i - \bar x)^2}}$。

这个 $t$ 统计量把"$\hat\beta_1$ 离真值多远"标准化了。利用它可以构造置信区间：

$\left[\hat\beta_1 - t_{1-\alpha/2}(n-2) \cdot \hat\sigma_{\hat\beta_1}, \quad \hat\beta_1 + t_{1-\alpha/2}(n-2) \cdot \hat\sigma_{\hat\beta_1}\right]$

生活语言：95% 置信区间意味着"如果我们反复做很多次同样的实验，每次都按这个方法算区间，大约 95% 的区间会包含真实的 $\beta_1$"。它**不是**说"真实的 $\beta_1$ 有 95% 的概率在这个区间里"。

Deutsch: Unter Normalverteilung gilt $\frac{\hat\beta_1 - \beta_1}{\hat\sigma_{\hat\beta_1}} \sim t_{n-2}$. Daraus folgt ein Konfidenzintervall zum Niveau $1 - \alpha$:

$\hat\beta_1 \pm t_{1-\alpha/2}(n-2) \cdot \hat\sigma_{\hat\beta_1}$

#### 同理对 $\beta_0$

$\frac{\hat\beta_0 - \beta_0}{\hat\sigma_{\hat\beta_0}} \sim t_{n-2}, \qquad \hat\sigma_{\hat\beta_0} = \sqrt{\hat\sigma^2 \cdot \frac{\sum_{i=1}^n x_i^2}{n \sum_{i=1}^n (x_i - \bar x)^2}}$

#### 逻辑链条总结

中文：整个推导链条是：

$\epsilon_i \sim N(0, \sigma^2)$ 独立 → $Y_i \sim N(\beta_0 + \beta_1 x_i, \sigma^2)$ 独立 → $\hat\beta_1$ 是 $Y_i$ 的线性组合 → $\hat\beta_1 \sim N(\beta_1, V(\hat\beta_1))$ → 标准化后是标准正态 → 用 $\hat\sigma^2$ 替换 $\sigma^2$（独立的 $\chi^2$）→ 得到 $t_{n-2}$ 分布 → 构造置信区间。

Deutsch: Die Kette lautet: $\epsilon_i$ i.i.d. normal → $Y_i$ unabhängig normal → $\hat\beta_1$ als Linearkombination ist normal → standardisiert ergibt $N(0,1)$ → Ersetzen von $\sigma^2$ durch $\hat\sigma^2$ (unabhängiges $\chi^2$) → $t_{n-2}$ → Konfidenzintervall.

#### R 练习 / R-Übung

```r
fit_cars <- lm(dist ~ speed, data = cars)

# summary 输出中关键数字
summary(fit_cars)
# Residual standard error: σ̂ ≈ 15.38
# speed 行: Estimate = 3.932, Std. Error = 0.4155, t = 9.464, p < 2e-16

# 置信区间 / Konfidenzintervall
confint(fit_cars, level = 0.95)

# 手动验证斜率的 95% CI
b1 <- coef(fit_cars)["speed"]
se_b1 <- summary(fit_cars)$coefficients["speed", "Std. Error"]
df <- fit_cars$df.residual   # n - 2 = 48
t_crit <- qt(0.975, df)
c(lower = b1 - t_crit * se_b1, upper = b1 + t_crit * se_b1)

# σ̂² 手动计算
sum(residuals(fit_cars)^2) / (nrow(cars) - 2)
sigma(fit_cars)^2   # 应该一样
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：如果把样本量从 50 增加到 500，置信区间大概会怎样变化？
   Deutsch: Wie ändert sich das Konfidenzintervall, wenn man statt $50$ nun $500$ Beobachtungen hat?
   **答案 / Antwort**：标准误大约变为原来的 $1/\sqrt{10}$，置信区间宽度大约缩为原来的 $1/\sqrt{10} \approx 0.32$ 倍——更多数据 = 更精确的估计。

2. 中文：如果 p 值非常小（如 $< 0.001$），是否意味着 $\beta_1$ 一定很大？
   Deutsch: Bedeutet ein sehr kleiner p-Wert (z.B. $< 0{,}001$), dass $\beta_1$ groß ist?
   **答案 / Antwort**：不一定。p 值小说明有充分证据认为 $\beta_1 \neq 0$，但效应可能很小——当 $n$ 很大时，即使很小的 $\beta_1$ 也能被"检测到"。效应大小要看 $\hat\beta_1$ 本身和置信区间。

### 2.7 知识点 6：平方和分解与 $R^2$——"模型拟合得好不好？" / Quadratsummenzerlegung und $R^2$

#### 核心分解（Folie 40）

中文：定义拟合值 $\hat Y_i = \hat\beta_0 + \hat\beta_1 x_i$。则 $Y$ 的总变异可以精确分解为两部分：

$\underbrace{\sum_{i=1}^n (Y_i - \bar Y)^2}_{SST} = \underbrace{\sum_{i=1}^n (Y_i - \hat Y_i)^2}_{SSE} + \underbrace{\sum_{i=1}^n (\hat Y_i - \bar Y)^2}_{SSM}$

| 缩写 | 全称 | 含义 中文 | Bedeutung Deutsch |
|---|---|---|---|
| SST | Sum of Squares Total | $Y$ 的总变异 | Gesamtstreuung von $Y$ |
| SSE | Sum of Squares Error | 模型没能解释的部分（残差变异）| Residuenstreuung |
| SSM | Sum of Squares Model | 模型解释的部分 | Vom Modell erklärte Streuung |

生活比喻：想象你的考试总分和平均分之间有差距（SST）。其中一部分差距可以用"学习时间"来解释（SSM），剩下的是其他因素造成的波动（SSE）。

Deutsch: Die Gesamtvariation $SST$ von $Y$ wird exakt in einen durch das Modell erklärten Teil $SSM$ und einen unerklärten Rest $SSE$ zerlegt.

#### 证明的直觉（Folie 43）

中文：把 $Y_i - \bar Y$ 拆成 $(Y_i - \hat Y_i) + (\hat Y_i - \bar Y)$，平方展开后交叉项 $C = \sum \hat\epsilon_i (\hat Y_i - \bar Y) = 0$（因为正规方程保证残差与拟合值正交）。所以 SST = SSE + SSM 精确成立。

#### 自由度（Folie 42）

中文：讲义解释了每个平方和的自由度（df）：

**SST 的 df = $n - 1$**：$n$ 个 $(Y_i - \bar Y)$ 受一个约束 $\sum (Y_i - \bar Y) = 0$，所以只有 $n - 1$ 个自由。

**SSE 的 df = $n - 2$**：$n$ 个残差 $\hat\epsilon_i$ 受两个约束（正规方程 $\sum \hat\epsilon_i = 0$ 和 $\sum x_i \hat\epsilon_i = 0$），所以只有 $n - 2$ 个自由。

**SSM 的 df = $1$**：$SSM = \sum (\hat\beta_1(x_i - \bar x))^2 = \hat\beta_1^2 \sum (x_i - \bar x)^2$，$x_i$ 已知后只有一个自由量 $\hat\beta_1$。

检验：$(n - 1) = (n - 2) + 1$ ✓

Deutsch: Die Freiheitsgrade sind: $\text{df}_{SST} = n - 1$, $\text{df}_{SSE} = n - 2$, $\text{df}_{SSM} = 1$.

#### $R^2$：决定系数（Folie 41, 44）

$R^2 = \frac{SSM}{SST} = 1 - \frac{SSE}{SST}$

中文：$R^2$ 表示"$Y$ 的总变异中有多少比例被模型解释"。

**简单线性回归的特殊性质**：$R^2 = r_{xY}^2$，即 Pearson 相关系数的平方。

$R^2$ 的性质（Folie 44）：

(a) $0 \le R^2 \le 1$。$R^2 = 1$ 意味着所有点完美落在直线上；$R^2 = 0$ 意味着 $x$ 完全不解释 $Y$ 的变异。

(b) $R^2$ 对 $x$ 和 $Y$ 的线性缩放不变——把速度单位从 mph 改成 km/h，$R^2$ 不变。

(c) $R^2$ 关于 $x$ 和 $Y$ 对称——回归 $Y \sim x$ 和回归 $x \sim Y$ 的 $R^2$ 相同。

(d) **注意**：$R^2$ 也取决于样本中 $x$ 的变异范围。如果你只测了 14–16 mph 的数据，$R^2$ 可能很低，但这不代表速度和距离无关——只是你的 $x$ 范围太窄了。

Deutsch: $R^2$ beschreibt den Anteil der Stichprobenvariation von $Y$, der durch das Modell erklärt wird. Wichtig: $R^2$ hängt auch von der Streuung der $x$-Werte ab; ein niedriges $R^2$ kann an einer engen $x$-Spanne liegen.

#### R 练习 / R-Übung

```r
fit_cars <- lm(dist ~ speed, data = cars)
y <- cars$dist
yhat <- fitted(fit_cars)

SST <- sum((y - mean(y))^2)
SSE <- sum((y - yhat)^2)
SSM <- sum((yhat - mean(y))^2)

c(SST = SST, SSE = SSE, SSM = SSM)
c(SSE_plus_SSM = SSE + SSM)   # 应等于 SST

R2 <- SSM / SST
c(R2_manual = R2,
  R2_summary = summary(fit_cars)$r.squared,
  cor_squared = cor(cars$speed, cars$dist)^2)
# 三者应相等，约 0.6511

# 自由度验证
c(df_SST = nrow(cars) - 1,
  df_SSE = fit_cars$df.residual,
  df_SSM = 1)
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：$R^2 = 0.65$ 在 `cars` 数据中意味着什么？
   Deutsch: Was bedeutet $R^2 = 0{,}65$ bei den `cars`-Daten?
   **答案 / Antwort**：刹车距离的总样本变异中，约 65% 可以被速度的线性模型解释，剩下 35% 是其他因素（路面、轮胎、反应等）和随机波动导致的。

2. 中文：如果你在一个全新的数据集中得到 $R^2 = 0.02$，这是否说明 $x$ 对 $Y$ 一定没有影响？
   Deutsch: Bedeutet $R^2 = 0{,}02$, dass $x$ keinen Einfluss auf $Y$ hat?
   **答案 / Antwort**：不一定。可能是 $x$ 的变异范围太窄、可能是关系是非线性的、也可能是误差确实很大。$R^2$ 低不等于"无关"。

### 2.8 知识点 7：一个真实例子——Wenigumstadt 考古数据 / Beispiel: Reihengräber in Wenigumstadt

#### 背景（Folie 39）

中文：讲义提到了 Wenigumstadt 考古遗址的数据：研究者测量了古代人骨的第 4 腰椎骨小梁厚度（$Y$）和死亡时年龄（$x$），想知道两者之间是否有线性关系。

这是一个典型的简单线性回归应用。讲义指出，这个例子的核心解读包括三个方面：(a) $\hat\beta_1$——年龄每增加一年，骨小梁厚度平均变化多少？(b) $\hat\sigma$——数据点围绕回归线的典型散布多大？(c) $R^2$——年龄能解释骨小梁厚度变异的多大比例？

同时也指出了该数据的特殊困难：测量误差（Messfehler）、数值很小（sehr kleine Werte）、以及 $x_i$（年龄）实际上也是估计出来的，不是真正"固定已知"的。

Deutsch: In den Daten der Reihengräber von Wenigumstadt werden Alter ($x$) und Knochenbälkchendicke ($Y$) untersucht. Die drei zentralen Größen der Interpretation sind $\hat\beta_1$, $\hat\sigma$ und $R^2$. Besonderheiten: Messfehler, sehr kleine Werte, und die $x_i$ (Alter) sind selbst geschätzt.

#### 为什么提这个例子？

中文：它展示了真实数据与"干净教科书数据"的差距——$x_i$ 不一定完美已知，测量有误差，值可能非常小。但线性回归仍然是一个有用的**第一近似**，前提是你清楚它的限制。

### 2.9 知识点 8：预测与预测区间 / Prognose und Prognoseintervall

#### 两个不同的问题

中文：给定一个新的 $x_{n+1}$（比如速度 = 22 mph），我们想知道：

**问题 A**："以 22 mph 行驶的所有车的**平均**刹车距离是多少？"——估计 $E(Y \mid x = 22) = \beta_0 + \beta_1 \cdot 22$。

**问题 B**："如果我现在以 22 mph 刹车，**这一次**刹车距离是多少？"——预测一个新的随机观测 $Y_{n+1}$。

问题 B 比问题 A 更不确定，因为它不仅有"我对直线位置的估计误差"，还多了一个"这次实验自身的随机波动 $\epsilon_{n+1}$"。

Deutsch: Bei einer neuen Beobachtung $x_{n+1}$ gibt es zwei Fragen: (A) Wie groß ist der **mittlere** $Y$-Wert? (B) Wo wird die **nächste einzelne** Beobachtung liegen? Frage B ist unsicherer, weil noch der neue Zufallsfehler $\epsilon_{n+1}$ hinzukommt.

#### 点预测（Folie 45）

$\hat Y_{n+1} = \hat\beta_0 + \hat\beta_1 x_{n+1}$

中文：两个问题的**点预测**是一样的——都是把 $x_{n+1}$ 代入估计直线。区别在于**不确定性**。

#### 预测误差方差（Folie 45）

$V(\hat Y_{n+1} - Y_{n+1}) = \sigma^2\left(\underbrace{1}_{\text{新误差}} + \underbrace{\frac{1}{n}}_{\text{均值不确定性}} + \underbrace{\frac{(x_{n+1} - \bar x)^2}{\sum_{i=1}^n (x_i - \bar x)^2}}_{\text{斜率不确定性}}\right)$

中文：这个公式有三项，每一项都有清楚的来源：

**第 1 项 = 1**：新观测自身的随机误差 $\epsilon_{n+1}$。即使你完美知道直线，下一次实验的刹车距离仍然是随机的。这一项在**预测区间**里有，但在**均值置信区间**里没有。

**第 2 项 = $\frac{1}{n}$**：对 $\bar Y$（直线整体高度）的估计不确定性。数据越多越小。

**第 3 项 = $\frac{(x_{n+1} - \bar x)^2}{\sum (x_i - \bar x)^2}$**：对斜率的估计不确定性。$x_{n+1}$ 离 $\bar x$ 越远，这一项越大——在"杠杆"远处，斜率的微小误差会被放大。

Deutsch: Die Varianz hat drei Terme: (1) neuer Zufallsfehler, (2) Unsicherheit des Gesamtmittelwerts, (3) Unsicherheit der Steigung, verstärkt durch die Entfernung von $\bar x$.

#### 预测区间（Folie 46）

$\hat Y_{n+1} \pm t_{1-\alpha/2}(n-2) \cdot \hat\sigma_{\hat Y_{n+1}}$

其中 $\hat\sigma^2_{\hat Y_{n+1}} = \hat\sigma^2\left(1 + \frac{1}{n} + \frac{(x_{n+1} - \bar x)^2}{\sum_{i=1}^n (x_i - \bar x)^2}\right)$。

#### 区间宽度的"沙漏形"

中文：如果你在不同的 $x$ 值处画置信区间和预测区间，会发现它们呈"沙漏形"——在 $x = \bar x$ 处最窄，越远离 $\bar x$ 越宽。预测区间始终比均值置信区间宽。

Deutsch: Die Intervalle bilden eine „Sanduhrform": Am schmalsten bei $x = \bar x$, breiter bei weiter entfernten $x$-Werten. Das Prognoseintervall ist immer breiter als das Konfidenzintervall für den Mittelwert.

#### R 练习 / R-Übung

```r
fit_cars <- lm(dist ~ speed, data = cars)
new <- data.frame(speed = seq(4, 30, by = 1))

ci <- predict(fit_cars, newdata = new, interval = "confidence")
pi <- predict(fit_cars, newdata = new, interval = "prediction")

plot(dist ~ speed, data = cars, xlim = c(4, 30), ylim = c(-50, 200),
     main = "Konfidenz- und Prognoseintervalle")
abline(fit_cars, col = "black", lwd = 2)

# 均值置信区间（窄）
lines(new$speed, ci[, "lwr"], col = "blue", lty = 2, lwd = 2)
lines(new$speed, ci[, "upr"], col = "blue", lty = 2, lwd = 2)

# 预测区间（宽）
lines(new$speed, pi[, "lwr"], col = "red", lty = 3, lwd = 2)
lines(new$speed, pi[, "upr"], col = "red", lty = 3, lwd = 2)

legend("topleft",
       legend = c("KQ-Gerade", "95% KI Mittelwert", "95% Prognoseintervall"),
       col = c("black", "blue", "red"), lty = c(1, 2, 3), lwd = 2)
# 观察：两对曲线都是沙漏形，预测区间始终更宽
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：为什么预测区间不可能比均值置信区间窄？
   Deutsch: Warum kann das Prognoseintervall nie schmaler sein als das Konfidenzintervall für den Mittelwert?
   **答案 / Antwort**：因为预测误差方差里多了 $\sigma^2 \cdot 1$（新观测自身的随机误差）。即使 $n \to \infty$，第 2、3 项趋于零，预测区间仍有宽度约 $\pm 2\sigma$。

2. 中文：在 $x_{n+1} = 30$ mph 处预测（超出数据最大速度 25 mph），你有什么担心？
   Deutsch: Was ist bedenklich, wenn man bei $x_{n+1} = 30$ prognostiziert, obwohl die Daten nur bis $25$ reichen?
   **答案 / Antwort**：这是**外推**（Extrapolation）。线性关系可能在 25 以上不再成立。即使预测区间看起来"正确"，它依赖的线性假设在那个范围没有数据支持。

### 2.10 知识点 9：线性模型作为近似与外推的风险 / Lineares Modell als Näherung und Extrapolationsrisiko

#### 讲义的数值例子（Folie 48–50）

中文：讲义设计了一个教学例子：真实关系是 $E(Y_i) = x_i^2$（抛物线），但我们只有 $x_1 = 0$、$x_2 = 1$、$x_3 = 2$ 三个点，并且错误地假设线性模型 $Y_i = \beta_0 + \beta_1 x_i + \epsilon$。

在这种情况下，$\hat\beta_1$ 的期望是：

$E(\hat\beta_1) = 2, \qquad E(\hat\beta_0) = -\frac{1}{3}$

在 $[0, 2]$ 的范围内，这条直线是抛物线的一个**可接受的线性近似**——尤其当误差 $\sigma$ 较大时，你可能根本看不出弯曲。

但在 $x = -1$ 处，直线预测 $\hat Y = -1/3 + 2 \cdot (-1) = -2\frac{1}{3}$，而真实值 $E(Y \mid x = -1) = (-1)^2 = 1$。**预测完全错误**，连方向都反了！

Deutsch: Wenn die wahre Beziehung $E(Y) = x^2$ ist und man irrtümlich ein lineares Modell annimmt, ist die Gerade im Bereich $[0, 2]$ eine akzeptable Näherung. Aber bei $x = -1$ ist die Prognose ($-2{,}33$) völlig falsch — der wahre Wert ist $1$.

#### 核心教训

中文：

(a) 线性模型在观测范围内可以是很好的**局部近似**（就像用直尺量弧度很小的弧线），但不保证在范围外也对。

(b) 外推越远，风险越大。预测区间在数学上会变宽（公式里 $(x_{n+1} - \bar x)^2$ 变大），但那个宽度假设的是**线性仍然成立**——如果关系实际上弯曲了，连区间本身都不可信。

(c) 如果对 $Y$ 或 $x$ 做变换（如 $\log Y$、$\sqrt{x}$），线性回归的工具仍然可用，但参数解释和误差假设都改变了。特别要注意 $E(g(Y)) \neq g(E(Y))$。

Deutsch: (a) Das lineare Modell ist oft eine brauchbare **lokale Näherung**, aber nicht automatisch außerhalb des Bereichs. (b) Extrapolation ist riskant; die Formeln gehen weiterhin von Linearität aus. (c) Transformationen ändern die Interpretation: $E(g(Y)) \neq g(E(Y))$.

#### R 练习：模拟讲义的例子 / R-Übung

```r
set.seed(42)
# 真实模型：Y = x² + ε
x <- c(0, 1, 2)
y <- x^2 + rnorm(3, sd = 0.3)

fit_lin <- lm(y ~ x)
fit_quad <- lm(y ~ x + I(x^2))

# 在 [-1, 3] 范围画图
x_plot <- seq(-1.5, 3, by = 0.01)
y_true <- x_plot^2
y_lin <- predict(fit_lin, newdata = data.frame(x = x_plot))

plot(x, y, xlim = c(-1.5, 3), ylim = c(-3, 10), pch = 19, cex = 1.5,
     main = "Lineare Näherung vs. wahre quadratische Funktion",
     xlab = "x", ylab = "y")
lines(x_plot, y_true, col = "black", lwd = 2)
lines(x_plot, y_lin, col = "red", lwd = 2)
legend("topleft",
       legend = c("wahre Funktion y = x²", "lineare Näherung"),
       col = c("black", "red"), lwd = 2)

# 外推到 x = -1
predict(fit_lin, newdata = data.frame(x = -1))
# 对比真实值 (-1)^2 = 1
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：为什么在 $[0, 2]$ 内直线近似还行，但超出就可能很糟？
   Deutsch: Warum funktioniert die Geradennäherung auf $[0, 2]$, aber nicht außerhalb?
   **答案 / Antwort**：就像用直尺量一个大圆上的一小段弧——弧度小时直线和弧线几乎重合，但延长直尺后它就会偏离圆弧越来越远。

2. 中文：如果你怀疑关系不是线性的，应该先做什么？
   Deutsch: Was sollte man tun, wenn man Nichtlinearität vermutet?
   **答案 / Antwort**：先画残差图 $\hat\epsilon_i$ vs. $x_i$——如果看到系统性弯曲，说明线性假设有问题。然后考虑变换（$\log$、$\sqrt{}$）或加多项式/样条项。

### 第 2 章总结 / Zusammenfassung Kapitel 2

**中文总结**：

- 简单线性回归模型 $Y_i = \beta_0 + \beta_1 x_i + \epsilon_i$ 把数据拆成**系统趋势**（直线）和**随机波动**（误差）。
- **最小二乘**通过最小化正方形面积总和找到最优直线，得到闭式公式 $\hat\beta_1 = S_{xY}/S_x^2$，$\hat\beta_0 = \bar Y - \hat\beta_1 \bar x$。
- 假设分三层：$E(\epsilon) = 0$ → 无偏；+独立同方差 → 方差公式；+正态 → $t$ 推断和置信区间。
- $\hat\beta_1$ 是 $Y_i$ 的加权和——这是理解无偏性、方差和正态性传递的钥匙。
- $\hat\sigma^2 = \frac{1}{n-2}\sum \hat\epsilon_i^2$，除以 $n - 2$ 是因为用掉了 2 个自由度。
- $SST = SSE + SSM$；$R^2 = SSM/SST = r_{xY}^2$，衡量线性解释的变异比例。
- 预测区间比均值置信区间宽（多了新误差 $\epsilon_{n+1}$），离 $\bar x$ 越远越宽。
- 线性模型是有用的局部近似，但外推有风险，变换改变解释。

**Deutsche Zusammenfassung**:

- Das Modell $Y_i = \beta_0 + \beta_1 x_i + \epsilon_i$ trennt systematischen Trend und zufällige Abweichung.
- Die KQ-Methode minimiert die Summe der Quadratflächen und liefert $\hat\beta_1 = S_{xY}/S_x^2$, $\hat\beta_0 = \bar Y - \hat\beta_1 \bar x$.
- Drei Annahme-Schichten: $E(\epsilon) = 0$ → Unverzerrtheit; + Homoskedastizität und Unabhängigkeit → Varianzformeln; + Normalverteilung → $t$-Inferenz.
- $\hat\beta_1$ ist eine gewichtete Summe der $Y_i$ — zentral für Unverzerrtheit, Varianz und Normalverteilung.
- $\hat\sigma^2 = \frac{1}{n-2}\sum \hat\epsilon_i^2$; die Division durch $n-2$ berücksichtigt die $2$ geschätzten Parameter.
- $SST = SSE + SSM$; $R^2 = SSM/SST = r_{xY}^2$, beschreibt erklärte Variation.
- Prognoseintervalle sind breiter als Konfidenzintervalle (neuer Fehler $\epsilon_{n+1}$), und weiter weg von $\bar x$ sind sie breiter.
- Das lineare Modell ist eine nützliche lokale Näherung; Extrapolation und Transformationen erfordern Vorsicht.
