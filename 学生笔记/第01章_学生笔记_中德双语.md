## 第 1 章：导论与例子 / Kapitel 1: Einführung und Beispiele

### 1.1 学习目标 / Lernziele

**中文**：这一章还不会推公式。目标是学会"看一眼就能识别回归问题"：谁是目标变量 $Y$，谁是解释变量 $x$，我们到底想解释、预测、还是检验某个效应。讲义通过五个生活化的例子（身高遗传、人生满意度、空气污染、阅读能力、慢性支气管炎）来说明：表面上完全不同的问题，可以放进同一套"回归建模"语言里。

**Deutsch**: Dieses Kapitel enthält noch keine Herleitungen. Das Ziel ist, ein Regressionsproblem zu erkennen: Was ist die Zielgröße $Y$, was sind die Einflussgrößen $x$, und geht es um Erklärung, Prognose oder das Testen eines Effekts? Die fünf Beispiele der Vorlesung zeigen, dass sehr unterschiedliche Fragestellungen in einer gemeinsamen Sprache der Regressionsmodellierung formuliert werden können.

### 1.2 知识点 1：回归到底在干什么？——一个生活比喻 / Was macht Regression? — Eine Alltagsanalogie

#### 先用一个最简单的比喻

中文：假设你想知道"多喝咖啡会不会让考试成绩变好"。你问了 100 个同学，记下每人每天喝几杯咖啡（$x$）和考试分数（$Y$）。你不会指望"喝了 3 杯的人分数全部一样"，因为成绩还受学习时间、天赋、心情等影响。但你可能会发现：喝 3 杯的人**平均**比喝 1 杯的人**平均**高一点。回归做的就是这件事——**找到 $Y$ 的平均趋势随 $x$ 变化的规律**，同时承认每个人围绕这个平均有随机波动。

Deutsch: Stell dir vor, du fragst 100 Studierende nach ihrem täglichen Kaffeekonsum ($x$) und ihrer Prüfungsnote ($Y$). Nicht alle, die 3 Tassen trinken, haben die gleiche Note — aber im **Mittel** unterscheiden sich die Gruppen vielleicht. Regression beschreibt genau diesen **mittleren Trend** von $Y$ in Abhängigkeit von $x$, wobei die zufällige Streuung um den Trend akzeptiert wird.

用数学写就是：

$Y = \underbrace{f(x)}_{\text{系统趋势 / systematischer Trend}} + \underbrace{\epsilon}_{\text{随机波动 / zufällige Abweichung}}$

其中 $E(Y \mid x) = f(x)$ 是"给定 $x$ 后 $Y$ 的平均值"。回归的任务就是从数据里学到 $f$ 的形状。

#### 为什么不只看相关系数？

中文：相关系数 $r$ 可以告诉你"$x$ 和 $Y$ 是否一起变动"，但它不区分谁是原因谁是结果（$r$ 对调 $x$ 和 $Y$ 后不变），也不能给你一个可以用来预测的公式。回归模型则明确指定"$Y$ 是目标、$x$ 是解释"，并且输出一个函数 $f(x)$，可以直接拿来做预测和检验。

Deutsch: Die Korrelation $r$ ist symmetrisch — sie ändert sich nicht, wenn man $x$ und $Y$ tauscht. Regression hingegen legt fest, dass $Y$ die Zielgröße ist, und liefert eine Funktion $f(x)$, die für Prognosen und Tests verwendet werden kann.

#### R 练习：相关系数 vs. 回归斜率 / R-Übung

```r
# 用 R 内置数据 women（15 个美国女性的身高和体重）
data(women)
head(women)

# 散点图 / Streudiagramm
plot(weight ~ height, data = women,
     main = "women: Gewicht vs. Körpergröße",
     xlab = "Körpergröße (inch)", ylab = "Gewicht (lb)")

# 相关系数 / Korrelation
cor(women$height, women$weight)   # r ≈ 0.9955
cor(women$weight, women$height)   # 交换后完全一样

# 回归：weight ~ height
fit1 <- lm(weight ~ height, data = women)
coef(fit1)  # Intercept ≈ -87.5, Slope ≈ 3.45

# 回归：height ~ weight（交换角色）
fit2 <- lm(height ~ weight, data = women)
coef(fit2)  # 斜率完全不同！

abline(fit1, col = "red", lwd = 2)
```

**直觉选择题 / Intuitionsfragen**：

1. 中文：`cor(x, y)` 和 `cor(y, x)` 一样，但 `lm(y ~ x)` 和 `lm(x ~ y)` 的斜率不同。为什么？
   Deutsch: `cor(x, y) = cor(y, x)`, aber die Steigungen von `lm(y ~ x)` und `lm(x ~ y)` sind verschieden. Warum?
   **答案 / Antwort**：相关系数衡量的是对称的共变关系，而回归斜率取决于"谁是 $Y$、谁是 $x$"——回归是在最小化 $Y$ 方向上的误差平方和，交换角色后优化方向变了。

2. 中文：高相关是否等于因果关系？
   Deutsch: Bedeutet hohe Korrelation automatisch Kausalität?
   **答案 / Antwort**：不等于。冰淇淋销量和溺水事件高度相关，但原因是夏天温度高。回归描述的是条件均值关系，因果需要研究设计或额外假设。

### 1.3 知识点 2：Galton 身高数据——"回归"一词的由来 / Galtons Daten — Woher kommt das Wort „Regression"?

#### 故事背景

中文：Francis Galton（1822–1911）收集了大量家庭的父母平均身高（$x$）和成年子女身高（$Y$）。他想看"高个父母的孩子是不是也特别高"。

Deutsch: Sir Francis Galton sammelte Daten über die mittlere Körpergröße der Eltern ($x$) und die Körpergröße ihrer erwachsenen Kinder ($Y$). Er wollte wissen, ob besonders große Eltern auch besonders große Kinder haben.

#### Galton 发现了什么？

中文：他拟合了一条直线（讲义 Folie 9）：

$E(Y) = \bar{Y} + 0.65 \cdot (x - \bar{x})$

化简后：

$E(Y) = 24 + 0.65 \cdot x$

关键数字是 **0.65**。如果身高完美遗传，斜率应该是 1（父母高 1 寸，孩子也平均高 1 寸）。但 0.65 < 1 意味着：**父母比平均高 1 寸，孩子平均只比平均高 0.65 寸**。极端高的父母，其孩子虽然也偏高，但没有父母那么极端——他们"回归"向了人群平均值。Galton 把这个现象叫做 **"regression toward mediocrity"（向平庸回归）**，这就是"回归"一词的历史来源。

Deutsch: Die geschätzte Gerade lautet $E(Y) = 24 + 0{,}65 \cdot x$. Die Steigung $0{,}65 < 1$ bedeutet: Wenn die Eltern 1 Zoll größer als der Durchschnitt sind, sind die Kinder im Mittel nur $0{,}65$ Zoll größer als der Durchschnitt. Extrem große Eltern haben zwar überdurchschnittlich große Kinder, aber nicht so extrem wie sie selbst — die Kinder „regressieren" zum Mittelwert. Galton nannte dies **„Regression toward mediocrity"**, und so entstand der Begriff **Regression**.

#### 用生活语言理解

中文：想象班里最高的同学 A 身高 190 cm。如果 A 的孩子长大后平均身高是 183 cm——还是很高，但没有 190 cm 那么极端。这就是"向均值回归"。反过来，班里最矮的同学 B 身高 155 cm，B 的孩子平均可能是 160 cm——还是偏矮，但没那么极端了。

Deutsch: Der größte Studierende im Kurs ist 190 cm groß. Sein Kind wird im Mittel vielleicht 183 cm — immer noch groß, aber nicht so extrem. Das ist Regression zur Mitte.

#### R 练习：用 R 内置数据模拟 Galton 的思路 / R-Übung

```r
# R 里没有 Galton 原始数据，但我们可以用 fathers.son 的思路
# 用 trees 数据来演示"斜率 < 1 的直觉"
data(trees)
# 想象 Girth（树围）类比"父母身高"，Height（树高）类比"孩子身高"
plot(Height ~ Girth, data = trees,
     main = "trees: 直觉上 Height 和 Girth 有关",
     xlab = "Girth (inch)", ylab = "Height (ft)")
fit_trees <- lm(Height ~ Girth, data = trees)
abline(fit_trees, col = "blue", lwd = 2)
summary(fit_trees)
# 观察斜率：Girth 每增加 1 inch，Height 平均增加多少 ft？
# 这个斜率的大小和方向是我们最关心的。
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：Galton 的斜率是 0.65。如果斜率恰好是 1，那意味着什么？如果是 0 呢？
   Deutsch: Die Steigung bei Galton ist $0{,}65$. Was würde eine Steigung von genau $1$ bedeuten? Was eine Steigung von $0$?
   **答案 / Antwort**：斜率 = 1 表示父母身高的偏差完全传给孩子（完美遗传）；斜率 = 0 表示父母身高对孩子身高的平均值没有任何影响（完全无关）。0.65 在两者之间：有影响，但会"打折"。

2. 中文："向均值回归"是否说明后代会越来越矮/越来越趋同？
   Deutsch: Bedeutet „Regression zur Mitte", dass spätere Generationen immer ähnlicher werden?
   **答案 / Antwort**：不是。这里说的是**条件均值**会比父母的极端值更靠近总体均值，但人群的整体方差并不会一代代缩小，因为每一代都有新的随机变异。

### 1.4 知识点 3：五个应用场景——每个问题对应一类模型 / Fünf Beispiele — Jede Frage führt zu einem Modelltyp

中文：讲义用 5 个例子说明，不同的研究问题自然地引导出不同类型的模型。下面逐一对照。

Deutsch: Die Vorlesung zeigt anhand von fünf Beispielen, wie unterschiedliche Fragestellungen zu unterschiedlichen Modelltypen führen.

#### 场景 A：Galton 身高（Folie 9）

| 项目 | 内容 |
|---|---|
| 研究问题 / Fragestellung | 父母身高如何影响孩子身高？ |
| 目标变量 $Y$ | 孩子身高（连续）/ Körpergröße des Kindes (stetig) |
| 解释变量 $x$ | 父母平均身高（连续）/ Körpergröße der Eltern (stetig) |
| 模型 / Modell | $E(Y) = \beta_0 + \beta_1 x$（**简单线性回归 / einfache lineare Regression**） |
| 生活理解 | 一个 $x$，一个 $Y$，画一条直线 |

#### 场景 B：人生满意度与年龄（Folie 10）

| 项目 | 内容 |
|---|---|
| 研究问题 / Fragestellung | 有没有"中年危机"？满意度和年龄是什么关系？ |
| 目标变量 $Y$ | 生活满意度（连续）/ Zufriedenheit (stetig) |
| 解释变量 $x$ | 年龄、收入、性别、健康状态等（多个变量混合）|
| 模型 / Modell | $E(Y_t) = \beta_0 + \beta_1 x_{1t} + \beta_2 x_{2t} + \beta_3 \log(x_{3t}) + \beta_4 x_{4t} + \dots + f(\text{Alter})$ |
| 模型类型 | **多元线性回归 + Splines / multiple lineare Regression + Splines** |
| 生活理解 | 不只一个影响因素了，而且年龄的效应可能是弯曲的（先降后升？），需要灵活的函数 $f$ |

中文补充：讲义图中，满意度在中年（约 40–60 岁）出现一个低谷，之后又上升——这正是"中年危机"的数据表现。但要注意，图中控制了收入、性别、健康等变量后的趋势，才是年龄本身的效应。

Deutsch: Die Grafik zeigt, dass die Zufriedenheit im mittleren Alter (ca. 40–60) ein Tief hat und danach wieder steigt. Wichtig ist, dass dies der Effekt des Alters **nach Kontrolle** von Einkommen, Geschlecht, Gesundheit usw. ist.

#### 场景 C：环境区与空气污染（Folie 11）

| 项目 | 内容 |
|---|---|
| 研究问题 / Fragestellung | 环境区（Umweltzone）的不同阶段是否降低了氮氧化物浓度？ |
| 目标变量 $Y$ | 每小时氮氧化物浓度 $NOX_t$（连续）/ Stündliche Stickoxide (stetig) |
| 解释变量 $x$ | 环境区阶段 LEZ(t) = 0,1,2,3（**分类变量**）+ 温度等连续变量 |
| 模型 / Modell | $E(NOX_t) = \mu(LEZ(t)) + \beta_1 \cdot temp + \dots$ |
| 模型类型 | **协方差分析 / Kovarianzanalyse (ANCOVA)** |
| 生活理解 | 既有"分组"（哪个阶段），又有连续变量（温度），两者同时放进模型 |

中文补充：$\mu(LEZ(t))$ 表示在环境区阶段 0、1、2、3 时的平均污染水平不同——这是一个分类变量的效应，和温度这种连续变量同时出现在模型里，所以叫"协方差分析"（ANOVA + 回归的结合）。

Deutsch: $\mu(LEZ(t))$ beschreibt unterschiedliche mittlere Belastungen in den Stufen 0, 1, 2, 3 der Umweltzone. Da gleichzeitig stetige Kovariablen wie Temperatur enthalten sind, spricht man von **Kovarianzanalyse**.

#### 场景 D：阅读能力与教学方法（Folie 12）

| 项目 | 内容 |
|---|---|
| 研究问题 / Fragestellung | 课外辅导时数是否提高阅读能力？不同班级有差异吗？ |
| 目标变量 $Y$ | 学生 $i$ 在班级 $j$ 的阅读能力 $Y_{ij}$（连续）|
| 解释变量 $x$ | 辅导时数 $x_{1ij}$（连续）+ 班级效应 $\alpha_j$（随机）|
| 模型 / Modell | $E(Y_{ij}) = \beta_0 + \beta_1 x_{1ij} + \alpha_j$ |
| 模型类型 | **混合线性模型 / Gemischtes lineares Modell (Linear Mixed Model)** |
| 生活理解 | 同一个班的学生之间不是完全独立的（同一个老师、同一个教室），所以需要给每个班一个"班级效应" $\alpha_j$ |

中文补充：为什么 $\alpha_j$ 是"随机的"？因为我们不关心"这个特定班级的效应是多少"，而是关心"班级之间的差异有多大"——$\alpha_j$ 被当作来自某个分布的随机抽样。

Deutsch: Warum ist $\alpha_j$ zufällig? Weil wir nicht an dem spezifischen Effekt einer bestimmten Klasse interessiert sind, sondern an der **Variabilität zwischen Klassen**. Die $\alpha_j$ werden als Zufallsvariablen aus einer Verteilung modelliert.

#### 场景 E：慢性支气管炎与粉尘暴露（Folie 13）

| 项目 | 内容 |
|---|---|
| 研究问题 / Fragestellung | 工作场所粉尘暴露是否增加慢性支气管炎的风险？ |
| 目标变量 $Y$ | 是否患慢性支气管炎（$Y_i = 0$ 或 $1$，**二元变量**）|
| 解释变量 $x$ | 粉尘浓度 $x_1$、暴露时长 $x_2$、吸烟状态 $IR$（是/否）|
| 模型 / Modell | $P(Y_i = 1) = G(\beta_0 + \beta_1 x_{1i} + \beta_2 x_{2i} + \beta_3 IR_i)$ |
| 模型类型 | **Logistic 回归 / Logistisches Regressionsmodell** |
| 生活理解 | $Y$ 不再是"多少"，而是"是或否"。普通线性回归会预测出小于 0 或大于 1 的概率，不合理；Logistic 回归通过一个 S 形函数 $G$ 把结果压在 $[0, 1]$ 之间 |

中文补充：$G$ 通常取 logistic 函数 $G(z) = \frac{e^z}{1+e^z}$。直觉上，$z$ 越大（粉尘越多、暴露越久、又吸烟），$G(z)$ 越接近 1，即患病概率越高。

Deutsch: $G$ ist typischerweise die logistische Funktion $G(z) = \frac{e^z}{1 + e^z}$. Je größer $z$ (mehr Staub, längere Exposition, Raucher), desto näher liegt $G(z)$ an $1$, also desto höher die Erkrankungswahrscheinlichkeit.

#### 五个场景的总览表 / Übersicht der fünf Beispiele

| 场景 | $Y$ 的类型 | $x$ 的类型 | 数据结构 | 对应模型类型 |
|---|---|---|---|---|
| A Galton 身高 | 连续 | 1 个连续 | 独立 | 简单线性回归 |
| B 中年危机 | 连续 | 多个连续+分类 | 独立 | 多元回归 + Splines |
| C 环境区 NOX | 连续 | 分类+连续 | 独立（时间序列特征） | 协方差分析 ANCOVA |
| D 阅读能力 | 连续 | 连续+班级分组 | 分组/嵌套 | 混合线性模型 |
| E 慢性支气管炎 | 二元 (0/1) | 连续+分类 | 独立 | Logistic 回归 |

```r
plot(Ozone ~ Temp, data = airquality,

# 这时 Y 是 0/1，普通线性回归不太合适

     xlab = "Temperature", ylab = "Ozone > 60? (0/1)")
```

#### 直觉选择题 / Intuitionsfragen

1. 中文：场景 C（环境区）里，如果不控制温度，直接比较不同阶段的 NOX 均值，可能有什么问题？
   Deutsch: Was kann passieren, wenn man in Beispiel C die Temperatur nicht kontrolliert und die NOX-Mittelwerte der LEZ-Stufen direkt vergleicht?
   **答案 / Antwort**：温度本身影响 NOX 浓度。如果某个阶段恰好在冬天实施（取暖排放高），那 NOX 偏高可能不是因为环境区无效，而是因为气温低。不控制温度会导致**混杂（Konfundierung）**。

2. 中文：场景 D 里，如果忽略班级效应 $\alpha_j$，直接把所有学生当独立样本做普通回归，标准误会怎样？
   Deutsch: Was passiert mit dem Standardfehler, wenn man in Beispiel D den Klasseneffekt $\alpha_j$ ignoriert?
   **答案 / Antwort**：标准误会被**低估**，因为同一班级的学生观测值之间正相关，实际的"有效样本量"比表面上的 $n$ 小。结果是 p 值偏小，容易产生假阳性。

### 1.5 知识点 4：建模前先做问题定义和初步数据分析 / Problemformulierung und erste Datenanalyse

#### 为什么不能一拿到数据就跑模型？

中文：想象你去医院，医生没问你哪里不舒服，没量体温，直接给你开刀——这显然不对。建模也是一样：先搞清楚问题，再看数据长什么样，最后才选模型。

Deutsch: Stell dir vor, ein Arzt operiert dich, ohne zu fragen, was dir fehlt. So ist es auch mit Modellen: Erst die Frage klären, dann die Daten anschauen, dann das Modell wählen.

#### 建模前的"四步检查清单" / Vier-Schritte-Checkliste

中文：拿到数据后，先回答以下四个问题：

| 步骤 | 中文 | Deutsch |
|---|---|---|
| 1. 目标变量 | $Y$ 是什么？单位是什么？连续、计数还是二元？| Was ist $Y$? Welche Einheit? Stetig, Zähldaten oder binär? |
| 2. 解释变量 | $x$ 有哪些？哪些连续，哪些分类？| Welche $x$-Variablen gibt es? Stetig oder kategorial? |
| 3. 研究目标 | 我要估计斜率、比较组均值、做预测，还是选变量？| Geht es um Steigungen, Gruppenvergleiche, Prognosen oder Variablenselektion? |
| 4. 数据质量 | 有没有缺失值、离群点、非线性？是否分组/重复测量？| Gibt es fehlende Werte, Ausreißer, Nichtlinearität, Gruppenstruktur? |

讲义里每个例子都是先说"研究问题"，再写出期望结构（比如 $E(NOX_t) = \mu(LEZ(t)) + \beta_1 \cdot temp + \dots$）。模型里的每一项都是从问题里来的，不是凭空添加的。

### 第 1 章总结 / Zusammenfassung Kapitel 1

**中文总结**：

- **回归的核心**是建模条件均值 $E(Y \mid x)$——不是说 $x$ 完全决定 $Y$，而是 $x$ 改变了 $Y$ 的平均趋势，每个个体围绕趋势有随机波动。
- **"回归"一词来自 Galton**：极端的父母，其子女会"回归"向人群均值（斜率 0.65 < 1）。
- **五个例子对应五种模型**：简单线性回归、多元回归+Splines、协方差分析、混合模型、Logistic 回归。变量类型和数据结构决定了选哪种。
- **建模前先做四步检查**：明确 $Y$、$x$、研究目标、数据质量。
- **课程路线**：从简单到复杂，第 2–4 章是核心，后续逐步放宽假设。

**Deutsche Zusammenfassung**:

- **Der Kern der Regression** ist die Modellierung von $E(Y \mid x)$ — nicht deterministische Bestimmung, sondern ein mittlerer Trend mit zufälliger Streuung.
- **Der Begriff „Regression" stammt von Galton**: Kinder extremer Eltern „regressieren" zum Mittelwert (Steigung $0{,}65 < 1$).
- **Fünf Beispiele führen zu fünf Modelltypen**: einfache lineare Regression, multiple Regression mit Splines, Kovarianzanalyse, gemischtes Modell, logistische Regression. Variablentypen und Datenstruktur bestimmen die Wahl.
- **Vor der Modellierung: Vier-Schritte-Checkliste** — $Y$, $x$, Ziel, Datenqualität klären.
- **Vorlesungsfahrplan**: Von einfach zu komplex; Kapitel 2–4 sind die Basis.
