## 1. 本章核心问题

第 1 章不急着推公式，而是先训练一种建模语言：

> 我们想用解释变量 $x$（Einflussgröße / Prädiktor）来理解或预测目标变量 $Y$（Zielgröße）。

在任何回归问题里，先回答四个问题：

| 问题 | 你要做什么 | 常用德语 |
|---|---|---|
| 谁是结果？ | 明确 $Y$ | Zielgröße, abhängige Variable |
| 谁解释结果？ | 明确 $x$ | Einflussgröße, Prädiktor, Kovariate |
| 目标是什么？ | 解释关系还是预测新值 | Erklärung, Prädiktion |
| 数据能不能直接建模？ | 看缺失、离群点、变量类型 | fehlende Werte, Ausreißer, Variablentypen |

本章最重要的总公式是：

$Y=f(x)+\epsilon$

其中 $f(x)=E(Y \mid x)$ 是系统趋势（systematische Komponente），$\epsilon$ 是随机误差或扰动项（Zufallsfehler / Störgröße）。

## 2. 从实际问题到回归问题

### 我遇到了什么问题？

SS26 讲义 Folie 5 给了很多应用场景：租金、医学、金融、环境、心理学。它们看起来不同，但统计翻译方式一样：先确定 $Y$，再确定 $x$。

| 场景 | $Y$ | $x$ |
|---|---|---|
| Mietpreise 租金 | Miete 房租 | Lage 位置、Größe 面积、Ausstattung 设施 |
| Medizin 医学 | Geburtsgewicht 出生体重 | Alter der Mutter 母亲年龄、Rauchen 吸烟 |
| Finanzen 金融 | Kreditrisiko 信用风险 | Einkommen 收入、Schulden 债务 |
| Umwelt 环境 | Waldzustand 森林状态 | Niederschlag 降水、Temperatur 温度 |
| Psychologie 心理学 | Testergebnis 测试成绩 | Alter 年龄、Bildung 教育 |

### 有哪些可能的解决路径？

| 路径 | 能解决什么 | 局限 |
|---|---|---|
| 描述统计 | 看均值、范围、分布 | 不能说明 $x$ 变化时 $Y$ 如何变化 |
| 相关系数 | 看两个变量是否共同变化 | 相关是对称的，不区分谁是 $Y$ |
| 回归模型 | 明确 $Y$ 和 $x$，学习 $E(Y \mid x)$ | 需要模型假设和诊断 |

### 为什么使用回归语言？

回归把不同问题统一到同一套框架中。比如“面积越大，租金是否更高”和“体重能否由身高预测”都可以写成：

$E(Y \mid x)=f(x)$

这句话的重点是“平均趋势”，不是说 $x$ 完全决定 $Y$。同样面积的房子租金可以不同，同样身高的人体重也可以不同。

### 怎么做？

1. 写一句研究问题，例如“汽车越重，油耗效率是否越低？”
2. 确定 $Y$：被解释或被预测的结果。
3. 确定 $x$：用于解释或预测 $Y$ 的变量。
4. 判断变量类型：连续、二元、计数、分类。
5. 决定目标：解释（Erklärung）还是预测（Prädiktion）。

### R 练习：识别 $Y$ 和 $x$

```r
data(mtcars)
str(mtcars)
summary(mtcars)

# 问题：汽车越重，油耗效率 mpg 是否越低？
plot(mpg ~ wt, data = mtcars,
     main = "mpg als Zielgröße, wt als Einflussgröße",
     xlab = "Gewicht wt (1000 lbs)",
     ylab = "Miles per gallon mpg")

fit_mpg_wt <- lm(mpg ~ wt, data = mtcars)
abline(fit_mpg_wt, col = "red", lwd = 2)
coef(fit_mpg_wt)

# 反过来建模时，问题变成“用 mpg 预测 wt”
fit_wt_mpg <- lm(wt ~ mpg, data = mtcars)
coef(fit_wt_mpg)
```

直觉检查：

- `lm(mpg ~ wt)` 中，`mpg` 是 $Y$，`wt` 是 $x$。
- `cor(mpg, wt)` 和 `cor(wt, mpg)` 一样，但 `lm(mpg ~ wt)` 和 `lm(wt ~ mpg)` 的问题不同。

## 3. SS26 租金例子：Nettomiete und Wohnfläche

### 我遇到了什么问题？

SS26 Folie 3-4 用慕尼黑租金例子开场：对 1966 年后建成、普通住宅位置的公寓，观察净租金（Nettomiete）和居住面积（Wohnfläche）的关系。

直觉上，面积越大，租金越高。但散点图不会是一条完美直线，因为同样面积的房子还可能因为设施、楼层、采光、噪音等原因有不同租金。

### 可能的解决路径

| 路径 | 含义 | 评价 |
|---|---|---|
| 只看平均租金 | 用一个数概括全部房子 | 太粗 |
| 按面积分组看均值 | 每个面积区间算平均租金 | 能看趋势，但不够简洁 |
| 画散点图 | 观察 $Y$ 和 $x$ 的关系 | 建模前必须做 |
| 拟合趋势函数 | 用 $E(Y \mid x)=f(x)$ 表示平均趋势 | 回归的核心 |

### 教授方法的好处

教授先用图，而不是直接上公式。这个顺序很好，因为散点图同时告诉我们两件事：

1. 系统趋势：面积越大，净租金平均越高。
2. 随机散布：同样面积附近仍有高低不同的租金。

教材把这件事写成：

$Y=f(x)+\epsilon$

在租金例子中：

- $Y=\text{Nettomiete}$
- $x=\text{Wohnfläche}$
- $f(x)=E(Y \mid x)$，表示给定面积时的平均净租金
- $\epsilon$，表示实际租金相对平均趋势的偏差

第 2 章会进一步把 $f(x)$ 先近似成直线：

$E(Y \mid x)=\beta_0+\beta_1x$

### R 练习：用 `trees` 类比“大小和价格”

R 没有内置慕尼黑租金数据。这里用 `trees` 练习同样的思路：树围 `Girth` 类比“面积/大小”，木材体积 `Volume` 类比“价格/租金”。

```r
data(trees)
str(trees)
summary(trees)

plot(Volume ~ Girth, data = trees,
     main = "Volume als Zielgröße, Girth als Einflussgröße",
     xlab = "Girth (inch)",
     ylab = "Volume")

fit_trees <- lm(Volume ~ Girth, data = trees)
abline(fit_trees, col = "red", lwd = 2)
coef(fit_trees)

# 残差：实际值 - 直线预测值
resid_trees <- residuals(fit_trees)
head(resid_trees)

plot(fitted(fit_trees), resid_trees,
     main = "Residuen gegen angepasste Werte",
     xlab = "Angepasste Werte",
     ylab = "Residuen")
abline(h = 0, col = "gray40", lty = 2)
```

直觉检查：

- 点围绕红线散开是正常的，因为回归建模的是 $E(Y \mid x)$，不是要求 $Y=f(x)$ 完全无误差。
- 如果残差图里还有明显形状，说明直线可能太简单，后续需要诊断（Modelldiagnose）。

## 4. 解释与预测：Erklärung vs. Prädiktion

### 我遇到了什么问题？

SS26 Folie 6 把线性建模目标分成两类：

- 解释（Erklärung）：$x$ 对 $Y$ 的平均值有什么影响？
- 预测（Prädiktion）：给定新的 $x$，如何预测新的 $Y$？

这两个目标经常被混在一起。比如：

| 问题 | 更偏向 |
|---|---|
| 屏幕时间更多的人是否平均睡得更少？ | 解释 |
| 身高 $1.60$ m 的人，体重大约是多少？ | 预测 |

### 为什么要区分？

解释和预测关注点不同：

| 目标 | 关注点 | 常见输出 |
|---|---|---|
| Erklärung | 变量方向、效应大小、不确定性 | 斜率、置信区间、检验 |
| Prädiktion | 新数据预测准不准 | 预测值、预测误差、预测区间 |

注意：预测准确不自动等于因果解释正确。回归首先描述的是条件均值关系 $E(Y \mid x)$；因果结论还需要研究设计和额外假设。

### R 练习：用身高预测体重

```r
data(women)
head(women)

plot(weight ~ height, data = women,
     main = "women: Körpergewicht und Körpergröße",
     xlab = "Height (inch)",
     ylab = "Weight (lb)")

fit_women <- lm(weight ~ height, data = women)
abline(fit_women, col = "red", lwd = 2)
coef(fit_women)
summary(fit_women)

# 预测身高 1.60 m 的人的体重
# 1 inch = 0.0254 m
new_person <- data.frame(height = 1.60 / 0.0254)
predict(fit_women, newdata = new_person, interval = "prediction")
```

操作提醒：

- `coef(fit_women)["height"]` 是斜率，用来解释身高每增加 1 inch，平均体重变化多少 lb。
- `predict(..., interval = "prediction")` 更偏向预测新个体。
- `women` 是很小的历史数据集，只适合课堂练习，不适合做现实健康判断。

## 5. 监督学习与条件均值

### 核心思想

SS26 Folie 9 说：我们从 $x$ 和 $Y$ 的数据中学习关系，这就是监督学习（Supervised Learning）。数据通常是一组观测对：

$(y_i,x_i),\quad i=1,\dots,n$

回归学习的不是“每个点都被完全命中”的确定性公式，而是条件均值：

$E(Y \mid x)=f(x)$

观测值可以拆成：

$Y=E(Y \mid x)+\epsilon=f(x)+\epsilon$

这里的 $\epsilon$ 表示随机波动。初学时最重要的直觉是：残差（Residuum）就是实际值和模型平均趋势之间剩下来的差。

### R 练习：用 `cars` 看系统趋势和残差

```r
data(cars)
head(cars)

plot(dist ~ speed, data = cars,
     main = "Bremsdistanz und Geschwindigkeit",
     xlab = "Speed (mph)",
     ylab = "Distance (ft)")

fit_cars <- lm(dist ~ speed, data = cars)
abline(fit_cars, col = "red", lwd = 2)

# 系统趋势的估计
head(fitted(fit_cars))

# 残差
head(residuals(fit_cars))

plot(fitted(fit_cars), residuals(fit_cars),
     main = "Residuen: Was bleibt nach dem Trend übrig?",
     xlab = "Angepasste Werte",
     ylab = "Residuen")
abline(h = 0, col = "gray40", lty = 2)

# 同一个 speed 下的平均 dist，用来感受 E(Y | x)
aggregate(dist ~ speed, data = cars, FUN = mean)
```

直觉检查：

- 点在回归线之上时，残差 $\hat\epsilon_i$ 为正。
- 点在回归线之下时，残差 $\hat\epsilon_i$ 为负。
- 残差如果不是随机散布，而是呈现曲线或漏斗形，后续要做模型诊断。

## 6. 建模前的初步数据分析

### 我遇到了什么问题？

教材和 R 补充书都强调：不要一拿到数据就直接跑模型。常见风险包括：

- 缺失值（fehlende Werte）
- 离群点（Ausreißer）
- 错误编码（falsche Kodierung）
- 变量单位理解错误
- 分类变量被当成连续变量

所以第 1 章要养成习惯：先看数据，再建模型。

### 建模前检查清单

| 步骤 | R 操作 | 目的 |
|---|---|---|
| 看结构 | `str(data)` | 变量类型是否正确 |
| 看范围 | `summary(data)` | 是否有不合理最小值/最大值 |
| 看缺失 | `colSums(is.na(data))` | 哪些变量缺失多 |
| 看单变量分布 | `hist()`、`density()`、`boxplot()` | 是否偏态、离群 |
| 看双变量关系 | `plot(Y ~ x)` | 是否线性、是否有明显结构 |
| 看分类分组 | `boxplot(Y ~ factor_x)` | 不同组的 $Y$ 是否不同 |

### R 练习：`airquality`

```r
data(airquality)
str(airquality)
summary(airquality)

# 每列缺失值数量
colSums(is.na(airquality))

# 单变量分布
hist(airquality$Ozone,
     main = "Histogramm von Ozone",
     xlab = "Ozone")

plot(density(airquality$Ozone, na.rm = TRUE),
     main = "Dichteschätzung von Ozone")

# 两个连续变量
plot(Ozone ~ Temp, data = airquality,
     main = "Ozone und Temperatur",
     xlab = "Temperature",
     ylab = "Ozone")

# Month 是数字编码，但作为月份组别时应转成 factor
airquality$Month_f <- factor(airquality$Month)
boxplot(Ozone ~ Month_f, data = airquality,
        main = "Ozone nach Monat",
        xlab = "Monat",
        ylab = "Ozone")

# 如果只用 Ozone 和 Temp，先看完整观测数量
aq_complete <- airquality[complete.cases(airquality[, c("Ozone", "Temp")]), ]
nrow(airquality)
nrow(aq_complete)
```

直觉检查：

- 如果 `Month` 代表组别，就不应简单把它当连续变量解释。
- `lm()` 会自动删掉含缺失值的行，但你必须知道有效样本量少了多少。

## 7. 变量类型决定模型方向

### 为什么重要？

模型选择首先看 $Y$ 的类型。不要把所有问题都硬塞进普通线性回归。

| $Y$ 类型 | 例子 | 常见模型方向 |
|---|---|---|
| 连续变量（stetig） | 租金、体重、测试成绩 | 线性回归：$E(Y \mid x)=\beta_0+\beta_1x+\dots$ |
| 二元变量（binär） | 是否违约、是否患病 | logistic 回归：$P(Y=1 \mid x)=G(\eta)$ |
| 计数变量（Zähldaten） | 事故次数、索赔次数 | Poisson 或计数回归 |
| 分类变量（kategorial） | 森林损伤等级、产品类别 | 分类模型或有序模型 |

### R 练习：连续 $Y$ 与分类 $x$

```r
data(ToothGrowth)
str(ToothGrowth)
summary(ToothGrowth)

# len 是连续 Y，supp 是分类 x
boxplot(len ~ supp, data = ToothGrowth,
        main = "Zahnwachstum nach Supplement",
        xlab = "Supplement",
        ylab = "Tooth length")

# dose 原本是数字剂量；如果把它当组别比较，转成 factor
ToothGrowth$dose_f <- factor(ToothGrowth$dose)
boxplot(len ~ dose_f, data = ToothGrowth,
        main = "Zahnwachstum nach Dosisgruppe",
        xlab = "Dosis",
        ylab = "Tooth length")

# 预告：连续 Y 可以用线性模型处理，分类 x 由 R 自动编码
fit_tooth <- lm(len ~ supp + dose_f, data = ToothGrowth)
summary(fit_tooth)
```

直觉检查：

- `supp` 是分类变量，不能像连续变量那样解释“增加一单位”。
- 分类变量进入线性模型时，R 会根据 factor 水平做编码；具体编码方式后面章节会学。

## 8. 教材补充：为什么叫“回归”？

SS26 第一讲没有重点展开历史，但主教材第 1 章从 Galton 的身高遗传数据讲起。

Galton 发现：父母身高很极端时，子女的平均身高也偏高或偏低，但通常没有父母那么极端，而是更靠近总体平均值。这叫向均值回归（Regression zur Mitte）。

标准化以后，简单回归有一个很有用的直觉：

$\hat z_Y=r z_x$

如果 $0<r<1$，那么 $x$ 高于平均 $2$ 个标准差时，预测的 $Y$ 通常高于平均不到 $2$ 个标准差。例如 $r=0.6$ 且 $z_x=2$，则：

$\hat z_Y=0.6\cdot 2=1.2$

注意：这不表示人群会一代代变得完全一样，只表示极端 $x$ 对应的条件平均 $Y$ 更靠近总体均值。

### R 练习：标准化回归直觉

```r
data(airquality)
aq <- airquality[complete.cases(airquality[, c("Ozone", "Temp")]), ]

aq$Temp_z <- as.numeric(scale(aq$Temp))
aq$Ozone_z <- as.numeric(scale(aq$Ozone))

fit_z <- lm(Ozone_z ~ Temp_z, data = aq)
coef(fit_z)
cor(aq$Temp_z, aq$Ozone_z)

plot(Ozone_z ~ Temp_z, data = aq,
     main = "Standardisierte Regression",
     xlab = "Temp_z",
     ylab = "Ozone_z")
abline(0, 1, col = "gray40", lty = 2, lwd = 2)
abline(fit_z, col = "red", lwd = 2)

predict(fit_z, newdata = data.frame(Temp_z = 1))
```

如果红线比灰色 $y=x$ 线更平，就说明预测值比 $x$ 的极端程度更靠近均值。

## 9. 本章总结

本章要掌握的是建模前的“翻译能力”：

1. 先确定目标变量 $Y$（Zielgröße）和解释变量 $x$（Einflussgröße）。
2. 回归的核心是条件均值 $E(Y \mid x)$，不是确定性公式 $Y=f(x)$。
3. SS26 的租金例子说明：面积影响平均租金，但实际租金仍有随机散布。
4. 线性建模有两个目标：解释（Erklärung）和预测（Prädiktion）。
5. 监督学习（Supervised Learning）意味着从成对数据 $(y_i,x_i)$ 中学习关系。
6. 建模前必须做初步数据分析：看结构、范围、缺失、图形关系。
7. 模型方向首先取决于 $Y$ 的类型：连续、二元、计数或分类。
8. “回归”一词来自 Galton 的向均值回归（Regression zur Mitte）现象。

## 10. 德语术语与公式

### 常用德语术语

| Deutsch | 中文 | 公式/备注 |
|---|---|---|
| die Zielgröße | 目标变量、响应变量 | $Y$ |
| die abhängige Variable | 因变量 | 通常同 $Y$ |
| die Einflussgröße | 解释变量、影响因素 | $x$ 或 $x_1,\dots,x_k$ |
| die erklärende Variable | 解释变量 | 同 Einflussgröße |
| die unabhängige Variable | 自变量 | 同 Einflussgröße，但不是因果保证 |
| der Prädiktor | 预测变量 | predictor |
| die Kovariate | 协变量 | covariate |
| die Regression | 回归 | 建模 $E(Y \mid x)$ |
| die Regressionsanalyse | 回归分析 | 系统研究 $Y$ 和 $x$ 的关系 |
| der Zusammenhang | 关系、关联 | Zusammenhang zwischen $Y$ und $x$ |
| die Erklärung | 解释 | 关注 $x$ 对 $E(Y \mid x)$ 的影响 |
| die Prädiktion | 预测 | 关注新观测的 $Y$ |
| die Vorhersage | 预测 | 同 Prädiktion |
| Supervised Learning | 监督学习 | 数据对 $(y_i,x_i)$ |
| der bedingte Erwartungswert | 条件期望 | $E(Y \mid x)$ |
| die systematische Komponente | 系统成分 | $f(x)=E(Y \mid x)$ |
| die stochastische Komponente | 随机成分 | $\epsilon$ |
| der Zufallsfehler | 随机误差 | $\epsilon=Y-E(Y \mid x)$ |
| die Störgröße | 扰动项 | 同误差项 $\epsilon$ |
| das Residuum | 残差 | $\hat\epsilon_i=y_i-\hat y_i$ |
| das Streudiagramm | 散点图 | 建模前观察 $Y$ 与 $x$ |
| das Histogramm | 直方图 | 单变量分布 |
| der Boxplot | 箱线图 | 常用于连续 $Y$ 与分类 $x$ |
| die fehlenden Werte | 缺失值 | R 中常见为 `NA` |
| der Ausreißer | 离群点 | 建模前检查 |
| die Kodierung | 编码 | 分类变量常需 factor 或 dummy coding |
| die stetige Variable | 连续变量 | 例如租金、体重、温度 |
| die kategoriale Variable | 分类变量 | 例如地区、月份、处理组 |
| die binäre Variable | 二元变量 | $Y\in\{0,1\}$ |
| die Regression zur Mitte | 向均值回归 | 标准化直觉：$\hat z_Y=r z_x$ |
| das einfache lineare Regressionsmodell | 简单线性回归模型 | $E(Y \mid x)=\beta_0+\beta_1x$ |
| die logistische Regression | logistic 回归 | $P(Y=1 \mid x)=G(\eta)$ |
| die Stichprobe | 样本 | $i=1,\dots,n$ |
| die Beobachtung | 观测 | 第 $i$ 个数据点 $(y_i,x_i)$ |

### 本章核心公式

| 作用 | 公式 | 解释 |
|---|---|---|
| 条件均值 | $E(Y \mid x)=f(x)$ | 给定 $x$ 时 $Y$ 的平均趋势 |
| 系统 + 随机 | $Y=f(x)+\epsilon$ | 观测值 = 平均趋势 + 随机偏差 |
| 误差定义 | $\epsilon=Y-E(Y \mid x)$ | 实际值偏离条件均值的部分 |
| 简单线性模型预告 | $E(Y \mid x)=\beta_0+\beta_1x$ | 第 2 章正式学习 |
| 预测 | $\hat Y_{\text{neu}}=\hat f(x_{\text{neu}})$ | 用估计趋势预测新观测 |
| 标准化回归直觉 | $\hat z_Y=r z_x$ | $|r|<1$ 时预测更靠近均值 |
| 二元响应预告 | $P(Y=1 \mid x)=G(\eta)$ | logistic 回归中使用 |
